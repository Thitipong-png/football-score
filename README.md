# GOAL! — World Cup 2026 Match Center

หน้าเว็บแสดงผลการแข่งขันฟุตบอลโลก 2026 แบบ responsive พร้อมตัวกรองสถานะแมตช์และ modal รายละเอียดการแข่งขัน ข้อมูลอัตโนมัติมาจากแผนฟรีของ football-data.org ผ่าน GitHub Actions โดย token ไม่ถูกส่งไปยังเบราว์เซอร์

## เปิดใช้งาน

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้ทันที หรือใช้ GitHub Pages สำหรับเว็บไซต์ออนไลน์

## การอัปเดตอัตโนมัติ

- ทำงานทุก 2 ชั่วโมงในช่วง 00:00–08:00 และ 18:00–22:00 ตามเวลาไทย
- ใช้ football-data.org Free Tier ประมาณ 1 request ต่อรอบ
- หยุดเรียก API ตั้งแต่ 00:00 วันที่ 22 กรกฎาคม 2026 ตามเวลาไทย
- ต้องสร้าง GitHub Actions secret ชื่อ `FOOTBALL_DATA_TOKEN`
- ใช้เฉพาะข้อมูลฟรี: โปรแกรมแข่งขัน สถานะแมตช์ และผลสกอร์ ไม่มีการเรียกข้อมูลแบบเสียเงิน

หน้าเว็บอ่านข้อมูลจาก `data/world-cup.json` และจะใช้ข้อมูลสำรองเมื่อเปิดจากไฟล์ในเครื่อง
